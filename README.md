### Features

- Upload and manage files (`.pdf`, `.docx`, `.txt`)
- Summarize documents using GPT-4 Turbo
- Ask contextual questions about file content
- Search within uploaded text
- Works with MongoDB or SQLite
- Deployable via Railway or Docker
